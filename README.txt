# HTML App

This is a simple HTML app created for demonstration purposes. It includes basic files for an HTML, CSS, and JavaScript setup.

## Files

- `index.html`: The main HTML file.
- `style.css`: Styles for the app.
- `script.js`: JavaScript for interactivity.
- `README.md`: Documentation for the project.

## How to Use

1. Open `index.html` in a browser to view the app.
2. Click the button to see the JavaScript functionality.

## Features

- Responsive design.
- Basic button interaction.
